## Key Implementation Details

**CopilotClient Lifecycle:** Singleton registered in DI, spawns one CLI process managing all sessions for application lifetime.

**Session Creation:** `CreateSessionAsync` loads system message (SOUL.md + bootstrap.md), creates SDK session with `InfiniteSessions = true`, returns session ID.

**Message Sending:** `SendMessageAsync` resumes session by ID, sends message via `SendAndWaitAsync` with 120s timeout, returns response.

**InfiniteSessions:** Enabled with SDK defaults (0.80 background compaction, 0.95 exhaustion threshold) for automatic context management.
