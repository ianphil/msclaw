# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MsClaw, please report it responsibly.

**Do not open a public issue.**

Instead, please email [ian.philpot@outlook.com](mailto:ian.philpot@outlook.com) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Supported Versions

| Version | Supported |
|---|---|
| Latest on `main` | ✅ |
| Older releases | ❌ |
